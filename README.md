# Roza Gift Corner & Electric - Customer Management System

A web-based customer and due management system for Roza Gift Corner & Electric shop.

## Features

- Customer management with name, phone, address, and source tracking
- Due and paid amount tracking with status indicators
- Advanced search and filtering capabilities
- Print and download receipts (customer and store copies)
- Dark/light mode toggle
- Data backup and restore functionality
- Password-protected sensitive operations
- Responsive design for mobile and desktop
